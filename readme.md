readme_content = """# Student Registration Portal

A lightweight web application for student registration. This project features a clean, stylized frontend form that processes student credentials through a PHP backend using local server integration.

---

## 🚀 Features

* **Dynamic UI Elements:** Real-time clock updated every second via JavaScript.
* **Intuitive Feedback:** Automatically reads URL parameters to display dynamic registration success or failure messages.
* **Two-Tier PHP Testing/Production Environment:**
    * **Test Server mode (`test_server2.php`):** Mock testing environment for verification without a database.
    * **Production Server mode (`server.php`):** Direct MySQL database integration for live registration logging.
* **Modern Responsive Footer:** Integrated social links for GitHub, Instagram, and Discord with active hover transitions.
